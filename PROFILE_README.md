## 🏆 Featured Project: Expedia Price Tracker

[![Chrome Web Store](https://img.shields.io/chrome-web-store/v/hkohddkblmjfdhlkhnbjoaekkncagcno?label=Chrome%20Web%20Store&style=for-the-badge)](https://chromewebstore.google.com/detail/expedia-price-scraper/hkohddkblmjfdhlkhnbjoaekkncagcno)
[![GitHub Repo](https://img.shields.io/badge/GitHub-Repository-100000?style=for-the-badge&logo=github&logoColor=white)](https://github.com/ParthPatel2000/expedia_price_tracker_extension)

A **production-grade Chrome extension** that automates hotel price tracking on Expedia with advanced features:

### ✨ Key Features
- 🤖 **Automated Price Scraping** with bot detection handling
- 📊 **Interactive Dashboard** with React & Recharts
- 📧 **Email Notifications** via Firebase Cloud Functions
- 🔐 **Google OAuth Integration** with anonymous fallback
- ⏰ **Configurable Scheduling** with Chrome Alarms API
- 💾 **Cloud Storage** with Firebase Firestore
- 🎨 **Modern UI** with TailwindCSS

### 🏗️ Architecture Highlights
- **Manifest v3** Chrome Extension APIs
- **React 19** for popup and dashboard
- **Firebase** ecosystem (Auth, Firestore, Cloud Functions)
- **Advanced scraping** with anti-bot measures
- **Production deployment** on Chrome Web Store

### 💼 What This Demonstrates
- **End-to-end ownership** from design to deployment
- **Scalable architecture** ready for team collaboration
- **Production experience** with Chrome Web Store
- **Advanced automation** and cloud integration
- **Real-world problem solving** for travelers
